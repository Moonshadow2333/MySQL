### 聚合函数
#### 介绍
> 将数据作为一个整体，进行纵向计算。

#### 常见聚合函数
|函数名|说明|
|:----:|:----|
|max|最大|
|min|最小|
|count|统计|
|sum|求和|
|avg|平均|

注意：**所有的 `null` 值不参与聚合函数运算。**

说明：count(*) 和 count(字段) 是有区别的，如果统计的字段中有些记录为空，就会统计所有记录不为空的记录总数，而不是总的记录数；

user 表如下：
|id|name|age|
|:----:|:----|:----:|
|1|Moon|18|
|2|小何|null|
|3|小白|18|

```
select count(*) from user; // 语句 1
select count(age) from user; // 语句 2
```
结果：
- 语句 1 得到的结果为 3；
- 语句 2 得到的结果为 2；
